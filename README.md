# Portofolio Fullstack Web Developer

Portofolio ini dibuat dengan React.js, Vite, dan Tailwind CSS. Desainnya modern, visual menarik, dan fokus pada presentasi profesional untuk HRD.

## Cara pakai

1. Buka terminal di folder `portofolio`
2. Pastikan Node.js sudah terpasang di sistem
3. Jalankan `npm install`
4. Jalankan `npm run dev`
5. Buka `http://localhost:5173`

> Jika halaman masih kosong setelah membuka browser, pastikan Anda menjalankan proyek melalui server Vite, bukan membuka `index.html` secara langsung.

## Struktur utama

- `src/App.jsx`: tata letak halaman portofolio
- `src/components`: komponen hero, tentang, proyek, kontak, dan footer
- `src/index.css`: gaya global Tailwind

## Kustomisasi

- Ganti teks pada `src/components` dengan detail pengalaman dan kontak Anda.
- Sesuaikan proyek di `src/components/Projects.jsx` sesuai hasil kerja nyata Anda.
- Tambahkan portofolio image atau link jika diperlukan.
